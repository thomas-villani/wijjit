# Security Policy

## Supported versions

Wijjit is pre-1.0. Security fixes are applied to the latest released version on
PyPI. Please make sure you are on the most recent release before reporting.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1.0 | :x:                |

## Reporting a vulnerability

Please **do not** open a public issue for security problems.

Instead, report privately using GitHub's
[private vulnerability reporting](https://github.com/thomas-villani/wijjit/security/advisories/new),
or email **thomas.villani@gmail.com** with:

- a description of the issue and its impact,
- steps to reproduce (a minimal example helps), and
- any relevant version/environment details.

You can expect an initial acknowledgement within a few days. Once the issue is
confirmed, we'll work on a fix and coordinate a disclosure timeline with you.
